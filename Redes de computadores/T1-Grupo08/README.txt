Para la correcta ejecución de la tarea (EN WINDOWS), se requerirá lo siguiente (y en este orden):

* *Abrir tres terminales distintas en la carpeta donde se encuentran los archivos* *

* *En una terminal ejecutar el comando : * *
	go run servidor_cachipun.go

* *En la siguiente terminal ejecutar:* *
	python servidor_intermediario.py

* *Y por último, en la restante:* *
	python cliente.py

El juego se desarrollará en cliente.py , aunque en el resto de las terminales se podrá
observar un flujo de mensajes. El resto de las instrucciones se encuentran al ya haber
ejecutado cliente.py

En caso de que se solicite windows firewall, se debe permitir.
